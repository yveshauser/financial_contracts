# Reference data

`bs.csv`, `crr.csv`, `jr.csv` and `tian.csv` are copied from the test suite
of [Miletus.jl](https://github.com/JuliaComputing/Miletus.jl) (MIT license).
They were generated by Miletus' `test/ref.r` using the R `fOptions` package:
European call/put prices (`c`, `p`) for the generalized Black-Scholes model,
and European/American call/put prices (`ce`, `pe`, `ca`, `pa`) for the CRR,
Jarrow-Rudd and Tian binomial trees, over a grid of start prices, strikes,
maturities, interest rates, carry rates, volatilities and step counts.
